# CCNA Network Labs

A personal portfolio of hands-on networking labs built while studying for the
Cisco CCNA certification track (ITN, SRWE, ENSA). Each lab uses my own
topology and addressing, not copied from coursework, and includes diagrams,
configurations, verification tests, and notes on design decisions.

I'm Arkadiusz Baran, an ICT student at AGH University in Krakow. This repo
is part of how I learn networking: by designing scenarios, configuring
devices in Cisco Packet Tracer / GNS3, and documenting what works and why.

## Tools used

- **Cisco Packet Tracer** - primary lab environment
- **GNS3** - for scenarios that need real Cisco IOS images
- **Wireshark** - packet capture and protocol analysis
- **Python + Netmiko** - automation labs (later in the ENSA track)
- **Git & GitHub** - version control and portfolio hosting

## How I learn

I study the theory through **Cisco Networking Academy (NetAcad)**, accessed
via AGH University. For each module I finish, I design a small lab that
covers the same concepts but with an original topology, different addressing,
different device names, different use case, so the lab is mine and not a
copy of the coursework.

Each lab follows the same structure: goal, topology, configuration highlights
with rationale, verification matrix, and lessons learned. The aim is that
anyone reading the lab folder can understand what was built and why, without
opening the `.pkt` file.
